## JSON ⇆ TOON parser

Workspace with the secure `json-toon-parser` library, unit tests, and an example project.

### Scripts

- `npm test` – run Vitest unit tests.
- `npm run build` – type-check and build the library.
- `npm run example` – run the example script after building.

### Structure

- `packages/json-toon-parser` – library source and README.
- `examples/basic` – minimal example wired to the workspace package.
